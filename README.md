# angularagrant

## AngularJS - Laravel - Vagrant 

Bash shell provisioner for Vagrant box to install Larabas-AngularJS development environment.

# Environment

- Node.js
  - chris-lea/node.js ppa for Ubuntu trusty
  - npm
  - gulp
  - Bower
  - recess

**WORK IN PROGRESS**

- Apache2 2.4.x
  - modrewrite
- Git
- PHP 5.6.x
  - mcrypt (php5-mcrypt)
  - intl (php5-intl)
  - sqlite (php5-sqlite)
- Composer
- Laravel 5.1
- AngularJS 1.4
